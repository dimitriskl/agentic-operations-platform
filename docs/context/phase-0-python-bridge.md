# Phase 0 — Python Bridge for C# Developer

## Goal

Learn only the Python needed to build agentic backend systems.

This phase is not about becoming a Python expert.

It is about becoming able to:

- read Python code
- modify Python code
- run Python code
- test Python code
- build small FastAPI services
- define Pydantic models
- understand project structure
- prepare for agent and tool development

## Day 1 target

Create a minimal FastAPI project with:

- virtual environment
- FastAPI app
- health endpoint
- Pydantic response model
- pytest test
- README
- learning log

## Day 1 definition of done

- project exists
- virtual environment works
- dependencies are installed
- FastAPI runs
- health endpoint returns JSON
- pytest passes
- README updated
- learning log updated

## C# analogies

FastAPI app is similar to an ASP.NET Core WebApplication.

Pydantic model is similar to a DTO or record with validation.

Route decorator is similar to a controller route attribute.

pytest is similar to xUnit or NUnit.

venv is an isolated dependency environment for one project.
